# Number Manipulation Functions – SQL Practice

In this project, I practiced MySQL numeric and mathematical functions using employee salary data.

## Functions Practiced

* `ABS()`
* `ROUND()`
* `CEIL()`
* `FLOOR()`
* `TRUNCATE()`
* `MOD()`
* `POWER()`
* `POW()`
* `SQRT()`
* `SIGN()`
* `RAND()`
* `FORMAT()`
* `GREATEST()`
* `LEAST()`

## What I Did

* Created SQL tables and inserted sample data
* Performed salary calculations using numeric functions
* Rounded salary values using `ROUND()`, `CEIL()`, and `FLOOR()`
* Removed decimal values using `TRUNCATE()`
* Calculated remainders using `MOD()`
* Used exponent and square root functions
* Formatted salary values for reports
* Compared maximum and minimum values using `GREATEST()` and `LEAST()`
* Practiced real-time SQL numeric function queries
