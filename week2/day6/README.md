# NULL FUNCTIONS IN MYSQL – SQL PRACTICE

In this project, I practiced MySQL NULL handling functions using Employees, Orders, and Products tables.

---

# Functions Practiced

* `IFNULL()`
* `COALESCE()`
* `NULLIF()`
* `IS NULL`
* `IS NOT NULL`

---

# Tables Used

## Employees
Contains employee salary, bonus, and manager details with NULL values.

## Orders
Contains order amount, discount, and coupon information with NULL values.

## Products
Contains product price, category, and stock details with NULL values.

# What I Did

* Created SQL tables with NULL values
* Practiced filtering NULL and NOT NULL records
* Replaced NULL values using `IFNULL()`
* Used `COALESCE()` to get first non-null values
* Used `NULLIF()` for comparisons and divide-by-zero handling
* Calculated salaries and payable amounts with NULL handling
* Solved real-time SQL NULL scenarios
